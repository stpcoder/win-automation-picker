# 문제 해결 확인 순서

문제가 생기면 기능을 다시 설정하기 전에 아래 순서로 원인을 좁힙니다.

1. 프로그램 상단의 `연결 확인`이 성공하는지 확인합니다.
2. 실장기 PC에서 `통신 중`인지 확인합니다.
3. 관리자 PC의 `마지막 상태 조회` 시각을 확인합니다.
4. `연결 구조 > 구성 검사`에서 오류가 있는지 확인합니다.
5. `이 PC COM 대조`로 COM과 USB 식별값을 확인합니다.
6. SK Commander에서 실장기 번호를 눈으로 확인합니다.
7. 상태 규칙의 `현재 값 읽기` 결과와 실제 화면 값을 비교합니다.
8. 실행 기록에서 마지막으로 완료된 블록을 확인합니다.
