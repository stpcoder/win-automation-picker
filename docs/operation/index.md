# 테스트 운용 순서

초기 설정이 끝난 뒤에는 테스트를 진행할 때만 아래 순서를 사용합니다.

1. `2 SEQ · 자동 실행 순서 준비`에서 SEQ 오류를 검사합니다.
2. 사용할 자동 실행 순서를 선택하거나 수정합니다.
3. `1 테스트 진행 > 테스트 실행`에서 실장기별 행을 불러옵니다.
4. 같은 SEQ를 사용할 실장기를 선택합니다.
5. 각 실장기의 장착 자재 ID와 필요한 입력값을 확인합니다.
6. `실행 시작`을 누릅니다.
7. `테스트 상태 보기` 또는 `실장기 상태`에서 진행 중, PASS, FAIL을 확인합니다.
8. 문제가 생기면 `긴급 중단`을 누릅니다.
9. 테스트가 끝나면 상태와 결과를 Excel로 내보냅니다.

![테스트 진행 화면](../assets/screenshots/01-test-run.png)

## 운용 원칙

- 같은 SEQ를 여러 TFT/UTF에 보낼 수 있습니다.
- 같은 SEQ를 보내더라도 장착 자재 ID는 실장기마다 다르게 유지됩니다.
- 관리자 PC에서 시작한 테스트와 실장기 PC에서 직접 시작한 테스트를 모두 상태 화면에서 확인합니다.
- 모니터링은 테스트를 진행하는 동안에만 켭니다.
- Binary 이름은 자동 추정하지 않고 마지막으로 확인한 사람이 직접 갱신합니다.
