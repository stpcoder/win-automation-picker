# 기본 매크로 만들기

## 클릭 블록 만들기

가장 빠른 방법:

1. 상단 `클릭 녹화`를 누릅니다.
2. 대상 프로그램에서 실제 버튼을 한 번 클릭합니다.
3. 가운데 작업실에 생긴 블록을 확인합니다.
4. 오른쪽 `이름`이 선택된 상태에서 `시작 클릭`처럼 알아보기 쉬운 이름을 입력하고 Enter를 누릅니다.
5. 작업실 위의 캡처 품질 배지가 `정상`, `확인 필요`, `재검토` 중 무엇인지 확인합니다.

원하는 위치에 바로 넣는 방법:

1. 왼쪽 `이벤트 > 클릭 녹화` 블록을 원하는 블록 사이로 끌어 놓습니다.
2. 대상 프로그램의 버튼을 클릭합니다.
3. 새 클릭 블록이 끌어 놓았던 위치에 생성됩니다.

## 입력 블록 만들기

1. 상단 `입력값`에 실제 문자열 또는 `${message}` 같은 변수를 입력합니다.
2. 기존 내용을 지울 경우 `기존값 지우기`를 켭니다.
3. 기본 방식은 `paste`입니다. 붙여넣기를 막는 프로그램이면 `keys`로 바꿉니다.
4. `입력 녹화`를 누르고 대상 입력칸을 클릭합니다.
5. 오른쪽 `이름`을 `serial_input`처럼 바꿉니다.
6. 필요하면 오른쪽 `입력할 텍스트`에서 블록별 값을 수정하고 `변경 적용`을 누릅니다.

!!! info "녹화할 때는 입력하지 않습니다"
    `입력 녹화`는 입력칸 위치만 캡처합니다. 실제 타이핑은 `실행`, `데이터 실행`, export한 Python 실행 시 발생합니다.

## 순서 바꾸기와 중첩

- 블록을 잡아 위아래로 끌면 실행 순서가 바뀝니다.
- `N번 반복`, `만약 ...` 블록의 빈 안쪽으로 다른 블록을 끌면 자식 블록이 됩니다.
- 자식 블록을 컨테이너 밖으로 끌어 다시 최상위 순서로 만들 수 있습니다.
- 드롭 가능한 위치에는 파란 삽입선이 표시됩니다.
- 블록을 더블클릭하면 오른쪽 이름 입력칸으로 바로 이동합니다.

## 실행

1회 실행:

1. 상단 `실행`을 누릅니다.
2. 비어 있는 반복/조건 블록이나 대상이 없는 블록이 있으면 실행 전에 해당 블록이 선택되고 오류 목록이 표시됩니다.
3. `실행 기록`에서 진행 순서와 오류를 확인합니다.

행별 실행:

1. `데이터 / 고급 > 데이터 행`에 Excel/Google Sheets 데이터를 붙여넣습니다.
2. 첫 행이 열 이름이면 `첫 행을 열 이름으로 사용`을 켭니다.
3. 상단 `데이터 실행`을 누릅니다.

## 저장과 복구

| 작업 | 방법 |
| --- | --- |
| 저장 | 상단 `저장` |
| 불러오기 | 상단 `불러오기` |
| Python 생성 | 상단 `Python 내보내기` |
| 직전 편집 취소 | 작업실 `되돌리기` 또는 `Ctrl+Z` |
| 취소한 편집 복구 | 작업실 `다시 실행` 또는 `Ctrl+Y` |
| 블록 복제 | 오른쪽 `복제` 또는 `Ctrl+D` |
| 블록 삭제 | 오른쪽 `삭제` 또는 Delete |
| 컨테이너만 제거 | 오른쪽 `풀기`; 내부 블록은 유지 |

## 잘못 캡처했을 때

1. 잘못된 블록을 선택합니다.
2. 캡처 품질 줄의 `선택 대상 다시 잡기` 또는 오른쪽 같은 이름의 버튼을 누릅니다.
3. 올바른 component를 다시 클릭합니다.
4. `선택 블록 시험`으로 한 블록만 확인합니다.
5. 같은 프로그램 창이 여러 개라면 `대상 고급 설정`에서 창 구분 조건을 추가합니다.
